  autocommit calls or a second wrapper transaction cannot overlap the
  open transaction
- added direct transaction-handle tests for commit/rollback,
  multi-mutator behavior, drop rollback, `inspect`, `renew`, and
  semantic parity
- updated wrapper docs and baseline docs to describe the sanctioned
  transaction path and its same-wrapper exclusivity
